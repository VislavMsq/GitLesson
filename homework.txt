1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
Открыть терминал или командную строку на своем компьютере
Зайти в папку, куда вы хотите скопировать репозиторий
Ввести команду: git clone и ссылку на свой репозиторий

2.Как создать git репозиторий из локального каталога на своем компьютере.
  Как его отправить на github.com?
Открыть терминал или командную строку на своем компьютере
Зайти в папку, которую вы хотите сделать Git репозиторием
Ввести команду: git init
Добавить файлы в репозиторий, используя команду: git add .
Сделать коммит, используя команду: git commit -m "Your commit message"
Создать репозиторий на GitHub, если вы еще этого не сделали
Скопировать URL репозитория на GitHub
Добавить удаленный репозиторий, используя команду: git remote add и ссылку на ваш репозиторий
Отправить изменения на GitHub, используя команду: git push -u origin master

3.Как получить на локальный компьютер изменения с github.com
Открыть терминал или командную строку на своем компьютере
Зайти в папку, которая является локальным репозиторием
Использовать команду: git pull

4.Какую информацию нам дают команда :
git status - показывает текущее состояние вашего локального репозитория Git, включая измененные файлы, неотслеживаемые файлы
git log - показывает историю коммитов в вашем локальном репозитории Git.
git remote -v - показывает список удаленных репозиториев Git, связанных с вашим локальным репозиторием
